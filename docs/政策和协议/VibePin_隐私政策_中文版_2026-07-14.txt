法律信息

隐私政策
最后更新日期：2026年7月14日

1. 概述

VibePin（以下简称“VibePin”“我们”）是一款面向电商商家的内容创建、规划与发布工作流平台。VibePin 帮助电商商家基于其拥有或控制的商品信息与内容，创建并管理适用于社交平台的商品内容。

本隐私政策说明我们收集哪些信息、如何使用和保护这些信息、如何处理与 Pinterest 有关的信息，以及用户可以行使的选择和权利。使用 VibePin 即表示您知悉本隐私政策所述的数据处理方式。

2. Pinterest API 与 OAuth 集成

VibePin 使用 Pinterest 官方 API 和 Pinterest OAuth，使经过授权的商家能够将其自己的 Pinterest Business 账号连接至 VibePin，并使用与 Pinterest 相关的功能。

连接 Pinterest 账号是可选操作。只有当用户主动发起 Pinterest 官方 OAuth 授权流程，并授予所请求的权限后，连接才会建立。VibePin 不会索取或存储用户的 Pinterest 密码。

VibePin 仅请求提供用户所选功能所合理需要的 Pinterest 权限。根据用户授予的权限，VibePin 可能通过 Pinterest API 获取有限的 Pinterest 账号信息、个人资料信息、画板信息、Pin 信息和发布相关信息，以支持账号连接、画板选择、内容审核以及经用户批准的发布。

VibePin 不会在缺少用户明确操作的情况下发布 Pin。发布前，用户可以审核内容、所选画板、目标网址（如有）及其他适用的发布设置，并且必须明确批准发布操作。

VibePin 不会使用 Pinterest API 自动执行互动、关注、评论、私信或其他人为制造的活动。VibePin 不会抓取 Pinterest，也不会将 Pinterest 数据用于未经授权的竞品监控。

VibePin 未获得 Pinterest 的认可、赞助，也不隶属于 Pinterest 或与 Pinterest 存在关联关系。Pinterest 是 Pinterest, Inc. 的商标。

3. 我们收集的信息

我们可能收集以下类别的信息：

账号信息：
用于创建和管理 VibePin 账号的信息，例如姓名、电子邮箱、账号标识符和认证信息。

商家及商品信息：
商品标题、描述、图片、商品网址、店铺信息，以及由商家导入或提供的其他内容。

内容信息：
通过 VibePin 创建、上传、编辑或批准的草稿图片、标题、描述、文案、链接、发布选择和其他内容。

Pinterest 相关信息：
当用户连接 Pinterest 时，我们可能根据 Pinterest API 以及用户授予的权限，接收有限的信息，例如 Pinterest 账号标识符、个人资料信息、画板信息、Pin 信息、发布状态及 OAuth 令牌。

使用及技术信息：
有关用户如何使用 VibePin 的信息，例如访问的页面、使用的功能、设备和浏览器信息、IP 地址、日志及错误信息。

通信信息：
用户在联系客服或与我们沟通时提供的信息。

4. 我们如何使用信息

我们使用信息用于：

- 提供、运营、维护和保护 VibePin；
- 验证用户身份并管理已连接的账号；
- 导入和整理商家提供的商品信息；
- 创建、保存、编辑和管理内容草稿；
- 显示 Pinterest 画板及其他经授权的账号信息；
- 仅在用户明确审核并批准发布操作后发布内容；
- 显示发布状态并协助诊断发布错误；
- 改进产品功能、可靠性、易用性和安全性；
- 回复支持请求并发送与服务有关的通知；
- 在法律允许的情况下发送产品更新或营销信息，并提供退订方式；
- 遵守法律义务并执行我们的条款和政策。

5. Pinterest 数据处理

Pinterest 相关信息仅用于提供用户明确请求并授权的功能。

VibePin 不会向第三方出售、出租、许可、转售、重新分发或以其他方式提供 Pinterest 内容或 Pinterest 衍生数据，供第三方独立使用、投放广告、从事数据经纪业务或用于其他无关目的。

我们不会使用 Pinterest 衍生数据为无关广告目的建立用户画像，也不会将 Pinterest 衍生数据与无关来源的数据合并后出售或重新分发。

我们可能使用代表 VibePin 处理信息的服务提供商，仅用于托管、保护、维护、监控和运营 VibePin。这些服务提供商不得将 Pinterest 相关信息用于其自身独立目的，并须遵守合同或法律规定的保密与数据保护义务。

6. 断开 Pinterest 连接及删除 Pinterest 相关数据

用户可以随时在 VibePin 设置中断开 Pinterest 账号，也可以通过 Pinterest 撤销对 VibePin 的授权。

Pinterest 账号断开连接或授权被撤销后：

- VibePin 将停止使用该项授权发起新的 Pinterest API 请求；
- 与该连接有关的 Pinterest OAuth 访问令牌和刷新令牌将被删除、失效或以其他方式变得不可用；
- 仅为提供 Pinterest 连接功能而存储的 Pinterest 衍生账号、画板、Pin 及发布信息，将在 30 天内删除或匿名化；
- 如为安全、防止欺诈、解决争议、遵守法律义务，或为提出、行使或抗辩法律权利而合理必要，我们可能保留有限的记录；
- 残余副本可能暂时存在于加密备份中，并通常在 90 天内根据正常的备份保留周期清除，法律要求保留更长时间的情况除外。

断开 Pinterest 连接不会自动删除用户独立创建或上传至 VibePin 的内容，例如商品信息或内容草稿；用户如需删除此类内容，应另行提出删除请求。

用户可通过 support@vibepin.co 联系我们，请求更早删除符合条件的 Pinterest 相关数据。

7. 数据共享及服务提供商

我们不会出售、出租或交易个人信息。

我们可能向协助运营 VibePin 的可信服务提供商披露有限的信息，包括云托管、数据库、身份验证、安全、分析、通信和客户支持服务提供商。这些服务提供商仅可为 VibePin 提供服务而处理信息，并须遵守适用的保密与数据保护义务。

我们还可能在以下情形披露信息：

- 法律、法规、法律程序或有效政府要求需要时；
- 为保护 VibePin、用户或其他人的权利、安全及系统完整性时；
- 与合并、收购、融资、重组或资产出售有关时，但会采取适当保护措施，并在法律要求时提供通知。

8. 数据保留

我们仅在提供 VibePin、实现本隐私政策所述目的、履行法律义务、解决争议及执行协议所合理必要的期间内保留个人信息。

账号信息和商家内容通常会在账号保持有效期间予以保留。用户可以随时请求删除其账号及相关的、符合删除条件的信息。

Pinterest 相关信息按照本政策第 6 节所述的断开连接和删除规则处理。

9. 数据安全

我们采取合理的管理、技术和组织措施，旨在防止信息遭到未经授权的访问、丢失、滥用、篡改或披露。

Pinterest OAuth 令牌通过适当的安全措施存储，不会向其他用户公开，也不会包含在公开的客户端代码中。但任何系统或传输方式都无法保证绝对安全，因此我们无法作出绝对安全保证。

10. 跨境处理

VibePin 及其服务提供商可能在用户居住国以外的国家或地区处理信息。在法律要求的情况下，我们将采用适当的跨境数据传输保障措施。

11. 用户的选择与权利

根据适用法律，用户可能享有以下权利：

- 请求访问我们持有的与其有关的个人信息；
- 请求更正不准确或不完整的信息；
- 请求删除符合条件的个人信息；
- 请求限制特定处理或反对特定处理；
- 请求获得某些信息的可携带副本；
- 在处理以同意为基础时撤回同意；
- 随时断开 Pinterest 连接或撤销 Pinterest 授权；
- 退订营销信息。

如需行使隐私权利，请联系 support@vibepin.co。在完成请求前，我们可能需要核实请求人的身份。

12. 儿童隐私

VibePin 不面向 13 岁以下儿童；如适用法律规定更高的最低年龄，则以该年龄为准。我们不会在违反适用法律的情况下故意收集儿童的个人信息。

13. 本隐私政策的变更

我们可能不时更新本隐私政策。如作出重大变更，我们将更新“最后更新日期”，并在法律要求的情况下提供额外通知。
